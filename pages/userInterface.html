User Interface

  1. What is a User Interface?

  2. A user interface is the point at which human users interact with a computer, website or application. It is what the user sees and how everything is laid out on the webpage.
  3. This interfers with how a user interacts with a webpage and how easy it may be for them.
  4. UI designers are responsible for the product's look and feel. UI designers can get creative and innovative and spend a lot of time to achieve an aestethically pleasing result.

User Interface

What do screen users want?

  • Orderly, clean, clutter-free appearance
  • Expected information located where it should be
  • For everything needed to be in front of the user
  • Should be suitable for beginner and advanced users

This leads to improved user production & increased satisfaction

What should a User Interface do?

A User Interface should make it easy for a user to accomplish what it is that they want to do

Is it more important that a website looks appealing or that the user finds it easy to use?

Jakob Nielsen : Designing Web Usability - "There are essentially two basic approaches to design: the artistic ideal of expressing yourself and the engineering ideal of solving a problem."

Computer User Interface Examples

    Windows

    Microsoft Windows was founded and introduced by Bill Gates on the 20th of November 1985. It is the most commonly used user interface by computer users as it is the most installed over nearly all computer platforms worldwide. It has developed through the years, has been updated many times for example Windows XP, 7, 8, 10 and the latest release Windows 11

    Mac OS

    Mac OS was founded and introduced by Steve Jobs in 1984 to run Apple's personal line of Macintosh computers. Apple's version of a computer user interface was mainly focused to be easy to use for the user and to achieve the highest productivity. They were supposed to be simplistic rather than having different commands and functions to navigate around.

    Linux

    All of the fastest supercomputers run Linux. Linux is ran on smartphones, spacerockets, submarines, servers, and a range of other devices. As of 2018, there are 20,323,379 lines of code in the Linux kernel.
