HyperText Markup Language

  1. What is HTML?
    • Stands for HyperText Markup Language.
    • Standard markup language for creating web pages.
    • Give structure to a webpage.
    • Consists of a series of elements, used to enclose/wrap different parts of the content to make it appear or act a certain way.
    • File consisting of "name.html".
  2. What is HyperText?
    • It is the text which links to other text or links to another page, example: www.ofppt.ma
  3. What is Markup?
    • It is the code used to give meaning to page content(text, images, videos)

HTML Tags

HTML tags are used to assign meaning to the page content. They are keywords described by the HTML standard and are designated by angle brackets. HTML tags are like keywords which define how the web browser will format and display the content. With the help of tags, a web browser can distinguish between different HTML content. HTML tags contain three main parts: opening tag, content and closing tag eg. There are exceptions to this though, as some HTML tags do not require a closing tag eg. <br> or <img> The current version of HTML is HTML 5

Examples Of Tags

    Tag Use
    <! doctype html> Tells the browser that this page is in HTML
    <html lang=“en”> In a valid page all content must be placed inside this element
    <meta charset> Tells the browser what character set to use
    <title> Appears as the browser window or tab title, also displayed in search engine results
    <body> Content to be displayed in the browser window <body>

    Block Elements

    <video> used to embed video content in a document, such as a movie clip or other video streams.

HTML Lists

  • An unordered list - achieved by using "ul" tags
  • Each item on the list is achieved by coding "li" tags inside "ul" tags
  • Useful if you don't need items to be listed in any order, like on a shopping list
  1. An ordered list - achieved by using "ol" tags
  2. Each item on the list is achieved by coding "li" tags inside "ol" tags
  3. Useful if items need to be listed in order and numbered, like dates or times
A detailed list - achieved by using "dl" tags
Each item on the list is achieved by coding "dt" tags inside "dl" tags
The detail/definition for each item is achieved by coding "dd" tags underneath "dt" tags
Useful for:
Lists that have more than one level of information or with items that have individual characteristics

HTML Iframe

An HTML iframe is used to display a web page within a web page.
HTML Iframe Syntax
The HTML <iframe> tag specifies an inline frame.
An inline frame is used to embed another document within the current HTML document.
Syntax
The <iframe src="url" title="description"></iframe>
Iframe - Set Height and Width
Use the height and width attributes to specify the size of the iframe.
The height and width are specified in pixels by default:
Example
<iframe src="../index.html" height="500" width="80%" title="description"></iframe>
